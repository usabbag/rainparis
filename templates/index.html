Command Line Interface

Check rain forecasts instantly from your terminal

1. Clone & Install

git clone https://github.com/yourusername/rainparis.git cd rainparis pip install -r requirements.txt

2. Add API Key

echo "TOMORROW_API_KEY=your_key" > .env

Get your free API key at tomorrow.io

3. Run

./rainparis 11

Or install globally: pip install -e .

Examples

rainparis 2 Quick check for 2nd arrondissement
rainparis 11 -v Detailed minute-by-minute forecast
rainparis --list Show all arrondissements
